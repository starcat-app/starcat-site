06账户与第三方凭证

6.1 GitHub 账户

使用本应用核心功能需要您拥有 GitHub 账户，并通过 OAuth 流程授权本应用以 read:user 与 public_repo 权限访问您的 GitHub 数据。本应用不会获取您的 GitHub 密码，也不会请求超出必要范围的权限。

6.2 凭证安全

GitHub OAuth Token 与您可选填写的 AI 服务 API Key 将仅保存在您本地 Apple 设备的系统 Keychain中，采用 Apple 提供的加密机制保护，并设置严格的 Access Control List。您有义务妥善保管您的设备、Apple ID 与生物识别凭证。

6.3 撤销授权

您可以随时在本应用中"设置 → 账户 → 退出登录"以删除本地存储的凭证，亦可在 GitHub 网站的"Settings → Applications"中撤销对 Starcat 的授权。
