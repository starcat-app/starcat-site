FAQ

Before you download

Starcat is currently focused on macOS. Here are the platform, data, and AI boundaries.

Which systems does Starcat support?

Starcat currently supports macOS 15 and later only. The Direct download is intended for Apple Silicon Macs. There are no iOS, iPadOS, watchOS, Windows, or Android builds today.

How is the Direct build different from an App Store build?

The landing page currently points to the Direct build, which supports in-app updates through Sparkle. App Store availability follows a separate release path and will be clearly labeled when available.

Where is my data stored?

Starcat is local-first. Repo cache can be rebuilt, while tags, notes, and status are stored in local SQLite on your Mac.

Does Starcat host my AI keys?

Starcat follows a BYOK model: you configure your own model provider or API key. AI-related features on this page depend on what is enabled in the current app build.

How do Pro license seats work?

Each Pro plan is licensed for up to 2 Macs you personally use. Pro Yearly saves 37% vs monthly. Lifetime is a one-time purchase that includes all Pro features with no recurring payments.

Will Starcat support iPhone or iPad later?

The short-term focus is making the macOS app stable and useful. Future platform plans will be announced in the changelog; this page does not promise multi-platform builds.
